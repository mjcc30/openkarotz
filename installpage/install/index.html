miniil's OpenKarotz Installer

This package will install miniil's OpenKarotz.
You don't have to flash again, and you don't have to use an USB key.
All files will be downloaded from karotz.miniil.be.
The installation will take about 2 minutes, after this you'll have the latest and most up to date miniil's OpenKarotz.
